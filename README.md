# 지금 나는 어떤 감정인가요

100여 개의 감정 단어 중에서 고르고, 또 내려놓으며 지금 내 안의 감정을 만나는 웹 서비스입니다.

🔗 **바로가기**: [https://hiation33-arch.github.io/emotion-journey](https://hiation33-arch.github.io/emotion-journey)

---

## 소개

감정을 단어로 표현하는 것이 어렵게 느껴질 때, 이 여정을 통해 지금 내 안에 흐르는 감정을 찾아볼 수 있습니다. "짜증나"로 퉁치던 감정들을 더 세밀하게 만나보세요.

## 흐름

1. **타임프레임 선택** — 오늘 / 요즘 / 자주 느끼는
2. **감정 고르기** — 긍정 감정 50개 중 최대 15개 선택
3. **덜어내기** — 8개까지 줄이기
4. **핵심 감정 찾기** — 3가지 덜어내어 남은 것 확인
5. **그림자 감정** — 외면했던 감정 5개 선택
6. **원하는 감정** — 각 그림자 감정의 반대편 선택
7. **결과** — AI 분석 프롬프트 복사 후 Claude/ChatGPT에 붙여넣기

## 특징

- 선택과 시간 데이터를 수집하여 AI 분석에 활용
- 모바일/데스크탑 반응형
- 외부 라이브러리 없이 순수 HTML/CSS/JS로 구현

## 기반 이론

- Robert Plutchik의 감정 바퀴 (Wheel of Emotions)
- Lisa Feldman Barrett의 감정 세분화 (Emotional Granularity) 이론
- PANAS (Positive and Negative Affect Schedule) 참고
